## 접근제한자

#### class 
- default
- public

#### feild, method
- public : 어느곳에서나 접근 가능
- private : 같은 클래스에서만 접근 가능
- protected : 상속 받으면 다른 패키지에서 접근 가능
- default : 같은 패키지 내 접근 가능

위 이미지는 protected를 구현함
